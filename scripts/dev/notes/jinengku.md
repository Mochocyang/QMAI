# 技能库分支说明

## 分支目标

实现一级入口「技能库」，第一版只管理多个去AI味技能，并接入 AI 会话、整章去AI味和选中文本去AI味。

## 使用要求

- 本分支只处理去AI味技能库，不扩展通用 Skill 库。
- 不删除旧版 `de-ai-skill.txt`。
- 不在本分支处理打包和 GitHub 推送。
